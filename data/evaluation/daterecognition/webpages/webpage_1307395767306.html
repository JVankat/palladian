Project Finance League Tables

Thomson Financial's standard league tables are based upon a single set of globally consistent criteria, submission guidelines, transaction classifications and eligibility rules, allowing you to accurately analyze transactions from around the world.

By clicking links, you can drill down to underlying deals data, all of which you can print or export to Excel.

 

To view Project Finance League Tables

  1. Select the Deals Analysis mode.

  1. Select Project Finance/League Tables, then select the Bookrunner Tables tab or the Mandated Arranger Tables tab.
    The Bookrunner Tables page appears by default when League Tables opens.

The Project Finance pages display the following tables:

  • International Corporate Bonds by Industry [Bookrunner page only]

  • Global Thomson/PFI League Tables (PDF Format) [Mandated Arranger page only]

  • Global Corporate Loans by Industry

  • Americas Corporate Loans by Industry

  • Europe, Middle East, Africa Corporate Loans by Industry

  • Asia Pacific Corporate Loans by Industry

  1. The Mandated Arranger Tables/Global Thomson/PFI League Table lists PFI (Project Finance International) League Tables on an annual basis and Thomson Financial Press Releases.
    Click any of the PDF icons to open the item described.

  2. For all other league table displays, select any LT Code (league table code) link to see a league table description.



    Note:     Click for access to recent and archived press releases.

  3. Select any Date Range link to drill down to the underlying deal activity report.
    Use the drop-downs to filter the displays according to Industry and time period.
