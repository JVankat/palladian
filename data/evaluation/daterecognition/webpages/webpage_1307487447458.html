Irie Pascal - Introduction

Teaching/Learning Pascal programming

The Irie Pascal compiler fully supports Standard Pascal (the "teaching" language) making it a great choice for use in courses like:

  • "Introduction to Programming" or
  • "Data Structures".

Irie Pascal is also a great choice, if you want to get back those Pascal programming skills you lost temporarily misplaced years ago.

Can I use Irie Pascal for business?

Yes, Irie Pascal adds many extensions to Standard Pascal (for working with files, strings, databases, etc) and these extensions make it pretty easy to write programs for what used to be called "data processing" or "batch processing".

For example, do you have a program that needs to read some data from a file, but that file is not in the correct format? Does someone (hopefully not you) have to manually edit the file to get into the correct format?

Well, Irie Pascal makes it pretty easy to write programs to automate these kinds of jobs.
